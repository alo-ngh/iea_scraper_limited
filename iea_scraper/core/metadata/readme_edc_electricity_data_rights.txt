To update the edc_electricity_data_rights.csv:
1. Modify and save the file
2. Run add_edc_data_rights.py in scripts\
	- You must use an env that has ieacore (https://python.iea.org/doc/ieacore/latest/)
3. Run I:\DW\EDC\00 - run\LoadEDC-DDS-Electricity Data Rights.bat

You can check the ETL has run here:
